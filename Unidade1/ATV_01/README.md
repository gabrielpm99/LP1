# Avaliação Zero

## Conteúdo do repositório

### Este repositório contém a resolução dos seis exercícios dispostos na Avaliação Zero. A lista está descrita abaixo:

- imprime_1aN.c (Imprime números de 1 a N);
- calcularFatorial.c (Calcula o fatorial de um número N);
- somaArray.c (Soma os números de um array de 5 números);
- conversorTemperatura.c (Converte a temperatura de Celsius para Fahrenheit);
- contaCaracteres.c (Conta a quantidade de caracteres de uma string);
- verificaPrimo.c (Verifica se um número N é primo).

## Requisitos

- Para rodar os programas, é necessário que tenha instalado e configurado a linguagem C e o compilador gcc em sua máquina.

## Como rodar

1. Abra o terminal da sua máquina ou do seu editor de código.

2. Pelo terminal, navegue até a pasta na qual se encontram os programas.

3. Para compilar o programa (por exemplo, somaArray.c), utilize o seguinte comando:

```bash
gcc somaArray.c -o somaArray.exe
```

4. Por fim, para rodar o programa (por exemplo, somaArray.exe), utilize este comando:

```bash
./somaArray.exe
```
